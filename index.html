🪶 HELIOS 🪶

RELEASE v1.2.1-b4085 ZeroTwo Cloud

Containerized Hyperconverged Infrastructure

Helios HCI is a shared-nothing software-defined datacenter stack built directly on host kernels. Co-locates metadata rings, SDN managers, consensus nodes, and storage routing inside systemd daemons and rootless containers.

Scaling Options: Deployable as a single-node hypervisor, a two-node HA pair, a three-node consensus cluster, or scaled out to any number of physical hosts. Replicates NVMe block segments synchronously over local host interfaces with native support for remote storage systems.

Lore & Cool Facts 🐉

  • Naming Scheme: Helios' architecture draws heavy inspiration from a mix of Norse and Zelda mythology!
  • Scale: The hyper-converged stack spans over 50,000 lines of pure Python, executing entirely on host kernel drivers.
