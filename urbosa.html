URBOSA

Layer-3 Software SDN Overlay
Urbosa is the lightning-fast brain behind Helios HCI's distributed networking. By utilizing advanced eBPF (Extended Berkeley Packet Filter) capabilities, it intercepts packets directly inside the Linux kernel before they even reach the standard networking stack!

Tier-0 Edge Gateway (T0)

The core North-South routing gateway. Operates in Active-Passive HA on the cluster leader node. It binds physical uplinks and orchestrates dynamic SNAT (Masquerading) and DNAT (port-forwarding) to effortlessly bridge external traffic into the overlay networks.

Tier-1 Distributed Router (T1)

The distributed East-West router. T1 namespaces are spawned identically across all hypervisor hosts. VMs communicate locally inside their host's kernel namespace without ever needing to egress to physical edge switches, unlocking zero-latency routing.

Integrated DHCP & IPAM

Urbosa completely automates IP address management. It injects a lightweight dnsmasq instance directly into the T1 logical router namespaces, securely issuing DHCP leases exclusively to VMs connected to authorized overlay segments.

VXLAN Overlay Fabric

Urbosa stretches L2 virtual segments seamlessly across physical boundaries. It pairs logical bridges with static point-to-multipoint VXLAN interfaces targeting peer host IPs, creating a massive full-mesh fabric on UDP port 4789.

Distributed Firewalls & eBPF

Security policies aren't enforced at the network edge; they are applied directly at the Virtual NIC of every single VM via stateful micro-segmentation. Instead of clunky iptables, Urbosa injects raw C-bytecode into the kernel for micro-second inspection without user-space context switches!
